# ASL Learning App

## Overview:
American Sign Language (ASL) is a rich and expressive visual language used by the Deaf and hard-of-hearing community. Learning ASL helps break down communication barriers, promotes inclusivity, and empowers more people to connect meaningfully across different abilities.

This project aligns with the United Nations Sustainable Development Goals (SDGs), in particular:

**SDG 4**: Quality Education – by making ASL learning more accessible and engaging through technology

**SDG 10**: Reduced Inequalities – by promoting inclusive communication and awareness of Deaf culture.

## What's Unique About This App:
While many existing ASL learning apps rely heavily on static quizzes and recognition-based learning, they often lack real-time feedback on actual signing. This limits a learner’s ability to develop proper muscle memory, which is essential for fluency in a visual-gestural language like ASL.

This app addresses this gap by integrating AI-powered live feedback, allowing users to actively practice signing and receive immediate, personalized guidance, thereby making learning more practical, engaging, and effective.

## Key Features:
**1. Engaging Homepage**
Dynamic ASL fun facts generated by AI
Designed to spark curiosity and keep users engaged with the language and culture

**2. Signing Skill Practice**
Users are given a randomly generated ASL letter to sign.
Practice is flexible as users can regenerate letters anytime and howeever many times they want to keep improving.

**3. Real-Time AI Feedback**
Users submit a photo of their sign for evaluation
AI analyzes the gesture and provides:
✅ Positive reinforcement when correct
❌ If incorrect:
AI identifies the sign the user actually made
and provides guidance on how to perform the correct sign

**Privacy-Focused Design**
No photos are stored, shared or saved
All image analysis is done securely with user privacy in mind.
