# ASL Learning App

## Overview:
American Sign Language (ASL) is a rich and expressive visual language used by the Deaf and hard-of-hearing community. Learning ASL helps break down communication barriers, promotes inclusivity, and empowers more people to connect meaningfully across different abilities.

This project aligns with the United Nations Sustainable Development Goals (SDGs), in particular:

SDG 4: Quality Education – by making ASL learning more accessible and engaging through technology

SDG 10: Reduced Inequalities – by promoting inclusive communication and awareness of Deaf culture.
